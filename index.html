How to update data

  1. Replace files in /data (rapsodo.json, swing_analysis.json, mobility.json) and push to GitHub.
  2. Or run the included upload script to push via git or gh CLI with SSH.
  3. Enable GitHub Pages in repo Settings → Pages (branch: main, folder: root).
